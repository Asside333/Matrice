Langue de travail : français. Toujours répondre en français.

## Identité

- **Nom :** MATRICE
- **Sous-titre :** "Calibre ton champ"
- **Symbole maître :** Cube de Métatron
- **Format :** PWA (Progressive Web App) — installable, hors-ligne, GitHub Pages
- **Langue interface :** français, mix FR/EN pour noms de modes
- **Design :** adaptatif (sombre avant 7h et après 19h, clair en journée)
- **Palette :** noir profond `#1A1A2E`, or ancien `#B8860B`, vert forêt `#0F6E56`, violet `#534AB7`, teal `#0F6E56` (SOS), cuivre `#B87333`
- **Typo :** Cormorant Garamond (sacré) + DM Sans (interface)

## Philosophie

MATRICE n'est pas une app de méditation. C'est un instrument de calibration personnelle. Le corps passe avant la pensée. Le sombre est un carburant, pas un ennemi. L'intention sans geste est vide.

## Géométrie sacrée

- Niveau 1 : Cube de Métatron (logo, accueil)
- Niveau 2 : Graine de Vie animée (respiration)
- Niveau 3 : 5 Solides de Platon = 5 éléments (Tétraèdre=Feu, Cube=Terre, Octaèdre=Air, Icosaèdre=Eau, Dodécaèdre=Éther)
- Niveau 4 : Spirale de Fibonacci (streak/progression)
- Symboles personnels : Plume de Maât, Yin Yang, 333 (easter eggs), Yi King (64 hexagrammes avec tirage 3 pièces)

## Modules

- **Rituel matin** : Souffle + Graine de Vie, Mantra, Intégration du sombre, Magnétisme, Sort du jour, Yi King + Clôture
- **Mode Libre / Studio** : binauraux + respiration à la volée
- **Mode SOS** : 3 phases — Corps, Nommer, Ancre
- **Mode Soir / Gratitude** : rapide ou complet

## Stack technique

HTML5 + CSS3 + JavaScript vanilla. Web Audio API pour binauraux. localStorage pour persistance. Service Worker pour offline + notifications. Déploiement GitHub Pages.
